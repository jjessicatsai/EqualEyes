# EqualEyes

## What is EqualEyes?
EqualEyes is a Chrome Extension that offers enhanced readability/accessibility for webpages. Features include Dark Mode, High Contrast Mode, Smart Contrast Mode, Enlarge Text, and Increase Text Spacing

## How to use
1. Download the `EqualEyes.zip` file
2. Extract the zip file in your file directory
3. In Google Chrome, search: chrome://extensions
4. In the top left, click "Load unpacked"
5. Upload the folder you created
6. The EqualEyes extension should appear under "All Extensions"
7. Pin the extension and try it out!
